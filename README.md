# Dra. Cynthia Netto — Site Institucional

Site institucional moderno para **Dra. Cynthia Netto de Barros**, MÉDICA ginecologista e mastologista em Belo Horizonte.

## Stack

- **Next.js 15** (App Router)
- **React 19** + TypeScript
- **Tailwind CSS 4**
- **Framer Motion** — animações
- **React Three Fiber** — visualizações 3D interativas nas especialidades
- **Vercel** — deploy

## Conformidade

Site desenvolvido seguindo o Relatório de Conformidade (CFM Res. 2.336/2023 + LGPD):

- Sem depoimentos ou avaliações de pacientes
- Identificação profissional com palavra **MÉDICA** e **CRM/MG**
- Copy informativo, sem promessas de resultado
- Política de Privacidade LGPD com Art. 11 (dados de saúde)
- Termos de Uso e banner de cookies com preferências granulares

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

### Anatomia 3D (Sketchfab)

Modelos 3D interativos via **Sketchfab** — embed gratuito, sem API key ou conta de desenvolvedor. Fonte: [AnatomyTOOL.org](https://anatomytool.org) (Catherine Vallance, Leiden University Medical Center, licença **CC BY**).

Alternativas open source para referência: [Z-Anatomy](https://www.z-anatomy.com) (CC BY-SA), [OpenAnatomy](https://openanatomy.org).

## Deploy

```bash
npm run build
```

Deploy automático via Vercel conectado ao repositório GitHub.
